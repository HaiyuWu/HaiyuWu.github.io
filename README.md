# Haiyu Wu — Personal Academic Webpage

A clean, responsive personal academic webpage built with plain HTML and CSS.

## Deploy to GitHub Pages (Free)

1. **Create a GitHub repository** named `<your-username>.github.io`
   (e.g., `haiyuwu.github.io`).

2. **Clone the repo locally:**
   ```bash
   git clone https://github.com/<your-username>/<your-username>.github.io.git
   cd <your-username>.github.io
   ```

3. **Copy all files** from this folder into the repo:
   ```bash
   cp -r /path/to/webpage/* .
   ```

4. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Initial site"
   git push origin main
   ```

5. **Your site is live** at `https://<your-username>.github.io` within a few minutes.

## Customizing

- **Profile photo**: Replace `assets/profile.jpg` with your own photo.
- **Content**: Edit `index.html` — each section is clearly marked with HTML comments
  (e.g., `<!-- PROFILE HEADER -->`, `<!-- ABOUT ME -->`, etc.).
- **Styling**: Edit `style.css` to change colors, fonts, spacing, etc.
- **Links**: Search for `REPLACE` or `#` in `index.html` to find placeholder URLs
  that need your actual links (Google Scholar, paper PDFs, etc.).

## Structure

```
index.html    — The entire single-page site
style.css     — All styles
assets/       — Images (profile photo, etc.)
README.md     — This file
```
